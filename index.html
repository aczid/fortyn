fortyn by kulp

A simulator for the Freescale HC08 series of 8-bit microcontrollers.

Fortyn is a simple simulator for the HC08 / S08 family of Freescale microprocessors. It is in alpha stage; all operations are implemented, but UIs are lacking, and testing is minimal.

Dependencies

Builds on Linux and probably other UNIXy systems using GCC. SDCC is required for the test suite. ncurses is required for the ncurses front-end.

Install

run "make"
